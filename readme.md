## Installation

1. **Create Python Virtual Environment (venv):**
   ```bash
   python -m venv venv
    ```
2. **Install Python requirements:**
    ```bash
   pip install -r requirements.txt
    ```
3. **Install JS requirements:**
   ```bash
   npm install $(cat npm_packages.txt)
   ```
